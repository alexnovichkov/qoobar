Qoobar Version

A free tag editor for classical music

Qoobar is a simple tagger which is designed for editing tags in files of classical music.

Qoobar uses Qt for GUI and Taglib for tags manipulation and runs under GNU/Linux, Mac OS X and Windows.

Qoobar also uses parts of other libraries: loudgain library, some files from Qt Creator sources and KDE sources.

Current version: Version, released under the GNU GPL license , that is Qoobar is absolutely free to use and modify.

Features

File types

  • ID3v2.4, APE, ASF, MP4 tags, Vorbis comments
  • MP3 (*.mp3), Ogg/Vorbis (*.ogg), Ogg/FLAC (*.oga), FLAC (*.flac), Musepack (*.mpc), Speex (*.spx), TrueAudio (*.tta), WavPack (*.wv), Wma/Asf (*.wma), APE (*.ape), Wav (*.wav), Aiff (*.aif, *.aiff), Opus (*.opus), MP4 (*.m4a, *.mp4), and DSF (*.dsf) files
  • More

Tags

Composer, Album, Title, Performer, Artist, Conductor, Orchestra, Subtitle, Key, Comment, Genre, Year, Track Number, Total Tracks, Album artist, Publisher, Mood, Beats per minute (Tempo), Original artist, Original lyricist, Lyricist, Original album, Lyrics, Music category (Grouping), Copyright, Disc number, Total discs, Remixed by, Encoded by, ISRC, URL, Rating

  • More

Editing

Qoobar allows you to

  • Generate tags from filenames and filepaths
  • Batch edit your files - edit any tags in group of files
  • Easy copy&paste any tags between groups of files
  • Remember the text you are typing for future use
  • Use autocompletion for selected tags
  • Add and remove cover art pictures
  • More

Other features

You also can

  • Import tags and covers from Discogs, Musicbraiz, GD3.
  • Rename / copy / move files by their tags, creating folders if necessary
  • Convert tags from local encoding into UTF8
  • Change the scheme of tags reading and writing
  • Add ReplayGain info into most of file types
  • Use Qoobar as a command-line utility (in Unix-like systems)
