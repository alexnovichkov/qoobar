Download and install

  Linux

The Qoobar project now uses openSUSE Build Service to build its packages. So it is much easier now to install Qoobar if you have one of the following systems:

  • Ubuntu 12.04, 14.04, 14.10, 15.04, 15.10
  • Fedora 20 - 22
  • CentOS 6 - 7
  • openSUSE 13.1 - 13.2
  • openSUSE Factory

The latest version can be downloaded from openSUSE software search:

  1. Type qoobar in the search line
  2. Click (search options) and check Include user's home projects
  3. Choose your system type and click Search.
  4. Choose the right architecture and download the package.

If you are using one of SUSE systems, feel free to use 1 Click Install.

  Ubuntu PPA

Ubuntu users can add this PPA repository: ppa:aleksej-novichkov/ppa. The installation process is as follows: 

> sudo add-apt-repository ppa:aleksej-novichkov/ppa
> sudo apt-get update
> sudo apt-get install qoobar

The supported Ubuntu versions: 12.04 (precise), 14.04 (trusty), 14.10 (utopic), 15.04 (vivid), 15.10 (wily). Other Ubuntu releases may contain outdated Qoobar packages, as they are no longer supported by PPA.

  Windows

Windows users can download the full 64-bit installer qoobar-version.exe from sourceforge.net.

Note: If you are encountering a problem with running Qoobar on Windows 7 or 8, you can perform these steps:

  1. Install the Update for Universal C Runtime
  2. Install the full Visual C++ 2015, 2017, 2019 and 2022 packages
  3. If nothing previous helped, install this version, compiled with the previous Qt version (5.15)

  Building Qoobar manually

Under Unix:
  1. Install Qt-devel >= 5.0, ffmpeg-devel, libswresample-devel, libebur128-devel, libtag1-devel, libdiscid, enca, shntool and gcc tools
  2. Download sources source files and unpack the archive or clone the Qoobar repo (see below).
  3. Go into the folder and in the console type: 
    $ qmake
$ make
$ sudo make install
$ make clean
Under Windows:
  1. Install Qt/Windows
  2. Download sources source files and unpack the archive or clone the Qoobar repo (see below).
  3. Download win_utils.tar.gz archive with necessary dlls and execs and unpack it into the source folder.
  4. In the console go into the source folder and type: 
    > qmake
> mingw32-make
> mingw32-make clean
  5. Copy release/* files to desired location or run release/qoobar.exe

The Git repository of Qoobar is https://github.com/alexnovichkov/qoobar.git. You can easily clone it with 

git clone https://github.com/alexnovichkov/qoobar.git

or with GitHub CLI:

gh repo clone alexnovichkov/qoobar
