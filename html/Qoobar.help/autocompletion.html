Autocompletion

Qoobar can autocomplete any tag while you are typing some text. This feature is turned on by default only for composers, genres, and moods. You can turn on autocompletion for other tags in the Preferences dialog.

Qoobar is shipped with three ready-for-use autocompletion lists: composer.txt, genre.txt and mood.txt. You can edit them in the Preferences dialog.

If you want Qoobar to remember some text for the future use, you have two options:

  • You can add the text manually. In the Tags Table select a tag or tags and press ⌘H or choose the context menu item Collect for future use. In the Edit Dialog select the text you want to remember and click Collect for future use.
  • You can also turn on the automatic collecting in the Preferences dialog. If this option is on, then any tag you type will be remembered automatically. This option is turned off by default.
