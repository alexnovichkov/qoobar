Menu

Qoobar

  • About Qoobar - shows a brief info about the current version
  • About Qt - shows the standard info about the Qt framework
  • Check for updates... - searches for new versions available
  • Preferences... - shows Qoobar Preferences dialog ⌘,
  • Exit - exits from Qoobar ⌘Q

File

  • Add folder... - adds a directory with all its subdirectories ⌘O
  • Add files... - adds a file / files ⇧⌘O
  • Split disc - splits an image of a musical disc (*.ape, *.flac, *.wav) by a cue sheet
  • Search files... - searches music files on your computer and places them in a separate tab ⇧⌘F
  • Save current tab - saves all changes in the current tab ⌘S
  • Save all tabs - saves all changes in all tabs
  • Reread tags - rereads tags in selected files and undoes all unsaved changes
  • Remove from the list - removes selected files from the current tab ⌦
  • Clear the list - clears the current tab ⌘⌦
  • Exit - exits from Qoobar ⌘Q

Edit

  • Undo last operation ⌘Z
  • Redo last operation ⇧⌘Z
  • Cut - cuts selected tags of selected files ⌘X
  • Copy - copies selected tags of selected files ⌘C
  • Paste - pastes copied tags ⌘V
  • Copy to clipboard - copies selected tags of selected files to the clipboard ⇧⌘C
  • Paste from clipboard - pastes copied tags from the clipboard ⇧⌘V
  • Clear all tags - clears all tags in the selected files
  • Add new tag... - adds a new tag to selected files ⌘N

Tools

  • Rename files... - renames / moves files according to its tags ⌘R
  • Fill tags... - fills tags from file name, tags or Internet ⌘F
  • Play selected - plays selected files in an external player ⌘P
  • ReplayGain info... - adds or removes ReplayGain info from files
  • Autonumber selected files... - adds or removes track numbers via the Autonumber dialog

Tabs

  • New tab - adds a new tab ⌘T
  • Close the tab - closes the current tab ⌘W
  • Close other tabs - closes all tabs except the current one
  • Rename the tab... - opens the dialog to rename the tab

Help

  • Help - shows help on Qoobar
