File types

Qoobar can read and write tags in different music files, the most common ones being mp3, m4a and flac files.

Each file type has its own unique icon that helps to differentiate them in the Files table.

Each music file type has its own set of tag types. The tag type describes the way in which tag information is written in a binary music file.
