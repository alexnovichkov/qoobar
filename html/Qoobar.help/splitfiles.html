Split Files

If you have a CD image (ape or flac or wav) and a cue file with it, Qoobar can split it to separate tracks, just click . You can also drag-and-drop a cue file into the Qoobar window.

Qoobar uses some external command-line programs. You can check if they are available in the Preferences Dialog.

If you have ffmpeg installed on your computer, you can choose the output files format. By default the format is flac, but you can set it to be m4a (ALAC codec), m4a (AAC codec), ogg, wma, wavpack, tta, speex, opus or mp3. I recommend to set 'alac' in Mac OS X and 'flac' in other systems, as they are lossless formats.

Note: By default all lossy formats are encoded with bitrate 320kb/s. If you want to change the bitrate, edit splitandconvert.sh or splitandconvert.bat script file and change the line "bitrate=320K" (or "set bitrate=320K").

Since ffmpeg is rather heavy, it is not shipped with the Qoobar installation. On Mac OS X you can install it by:

  1. using HomeBrew: 
    brew install ffmpeg
  2. using MacPorts: 
    sudo port install ffmpeg
  3. downloading it from ffmpegmac.net and placing the executable into qoobar.app/Contents/Resources/ folder.


The ordinary workflow is as follows:

  1. Choose the .cue file.
  2. Change (if needed) the output folder where the splitted files will be placed.
  3. Change the output files format (if available).
  4. Press Start.

All splitted files will be added to the end of the current tab.
