Toolbar

  1. Add folder... - adds a directory with all its subdirectories ⌘O
  2. Add files... - adds a file / files ⇧⌘O
  3. Split disc - splits an image of a musical disc (*.ape, *.flac, *.wav) by a cue sheet
  4. Save - saves all changes in the current tab ⌘S
  5. Reread tags - rereads tags in selected files and undoes all unsaved changes
  6. Undo / Redo last operation ⌘Z / ⇧⌘Z
  7. Rename files... - renames / moves files according to its tags ⌘R
  8. Fill tags... - fills tags from file name, tags or Internet ⌘F
  9. Clear all tags - deletes all tags in the selected files
  10. Clear the list - clears the current tab ⌘Del
  11. Play selected - plays selected files in an external player ⌘P
  12. Cut - cuts selected tags of selected files ⌘X
  13. Copy - copies selected tags of selected files ⌘C
  14. Paste - pastes copied tags ⌘V
  15. Add new tag... - adds a new tag to selected files ⌘N

You can customize the toolbar by right-clicking on it.
