Features

With Qoobar you can

  • Read and write standard tags: Composer, Album, Title, Performer, Artist, Conductor, Orchestra, Subtitle, Key, Comment, Genre, Year, Track Number, Total Tracks, Album artist, Publisher, Mood, Beats per minute (Tempo), Original artist, Original lyricist, Lyricist, Original album, Lyrics, Music category (Grouping), Copyright, Disc number, Total discs, Remixed by, Encoded by, ISRC, URL, Rating.
  • Add your own tags for all supported file types.
  • Read, write and import Cover art.
  • Generage tags from files names and other tags.
  • Move / copy / rename files.
  • Edit tags: Copy / paste, change case, recode, transliterate tags, etc.
  • Use placeholders in renaming, filling, editing operations.
  • Undo / redo any your action separately for each opened tab.
  • Import tags from Discogs, Musicbrainz and freedb databases.
  • Split a CD image by the corresponding CUE file.
  • Use Autocompletion for any tag.
  • Change Tagging scheme that defines how to read and write tags.
  • Add and remove ReplayGain info in Mp3, Mp4, Ogg Vorbis, Flac, Musepack, WavPack files.
  • Use Command-line interface in Unix-like systems.
  • Search for music files in standard and user defined locations.
