Appendix 2. Shortcuts

Shortcut Action
⌘AIn the files table: Select all files
In the tags table: Select all tags
⌘CCopy selected tags
⇧⌘CCopy selected tags into clipboard
⌘↓Move selected files down
⌘FIn the main window: Fill tags with the information from files names
In the tags editing dialog: Start search & replace
⇧⌘FSearch for music files on your computer
⌘HAdd selected tags to the autocompletion list
⌘NAdd a new tag to the selected files
⌘OAdd a folder with music files (with all subfolders)
⇧⌘OAdd a file / files
⌘PPlay selected files in an external player
⌘QExit from Qoobar
⌘RRename selected files
⌘SSave all files in the current tab
⌘TNew tab
⌘↑Move selected files up
⌘VPaste copied tags
⇧⌘VPaste text from clipboard into a selected tag
⌘WClose current tab
⌘XCut selected tags
⌘ZUndo the last change
⇧⌘ZRedo the last change
⌦In the files table: Remove selected files from the list
In the tags table: Delete selected tags
⌘⌦Clear the files table of the current tab
F1Help on Qoobar
F3In the tags editing dialog: Find next (in Windows)
⇧F3In the tags editing dialog: Find previous (in Windows)
